# Image_Editor_using_frontend

This is a WEBPAGE using HTML, CSS, JAVASCRIPT to edit uploaded images.
Included Features are : 
1] Brightness
2] Contrast
3] Saturation
4] Blur
5] Invert
6] Rotate
7] Flip Horizontal and Vertical
8] Zoom In and Out
9] Add Text
10] Add Emoji i.e Heart & Star
11] Crop
12] Reset
13] Download
14] Upload
